# ELM327 BT EEPROM Configurator

Утилита для настройки EEPROM Bluetooth-модулей, используемых в адаптерах **ELM327 OBD-II**.  
Позволяет изменять имя устройства, PIN-код, скорость UART, параметры автоподключения и другие настройки EEPROM.

## Возможности

- Изменение имени Bluetooth-устройства для BT2.0 и BLE
- Смена PIN-кода
- Смена MAC-адреса
- Настройка скорости UART
- Сохранение конфигурации в файл
- Работа с BT2.0 и BLE дампами



## Системные требования

- Windows x86

## Предупреждение

Некорректная запись EEPROM может вывести адаптер из строя.  
Используйте резервное копирование перед изменениями.
