# ELM327 BT EEPROM Configurator

Утилита для настройки EEPROM Bluetooth-модулей, используемых в адаптерах **ELM327 OBD-II**.  
Позволяет изменять имя устройства, PIN-код, скорость UART, параметры автоподключения и другие настройки EEPROM.

## Возможности

- Изменение имени Bluetooth-устройства для BT2.0 и BLE
- Смена PIN-кода
- Смена MAC-адреса
- Настройка скорости UART
- Сохранение конфигурации в файл
- Работа с BT2.0 и BLE дампами


## Назначение

Многие адаптеры ELM327 поставляются с неудобными параметрами:

- одинаковое имя (`OBDII`, `ELM327`, `V-LINK`)
- стандартный PIN (`1234`, `0000`)
- нестабильная скорость обмена
- проблемы совместимости с Android / Windows

Программа позволяет быстро адаптировать устройство под свои задачи.

## Интерфейс

Простой графический интерфейс:

- поиск COM-порта
- подключение к модулю
- чтение EEPROM
- редактирование параметров
- запись настроек

## Применение

Подходит для:

- автодиагностики
- ремонта ELM327
- кастомизации Bluetooth-адаптеров
- восстановления нерабочих клонов

## Системные требования

- Windows 10 / 11
- Bluetooth USB Dongle или встроенный Bluetooth
- COM-порт устройства
- .NET / Delphi Runtime (в зависимости от сборки)

## Использование

1. Подключите ELM327 к питанию.
2. Выполните сопряжение через Bluetooth.
3. Узнайте COM-порт.
4. Запустите программу.
5. Подключитесь.
6. Считайте EEPROM.
7. Измените параметры.
8. Нажмите **Write**.

## Предупреждение

Некорректная запись EEPROM может вывести адаптер из строя.  
Используйте функцию резервного копирования перед изменениями.
